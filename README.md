# cafeteria
Un diseño sencillo para una página de blogs de una cafeteria.

Como siempre el index es la pagina inicial y contiene un enlace a cada una de las demas paginas.
Todos los archivos html solo estan enlazados a un unico archivo css llamado "style.css" desde el cual se importaron todos los demas archivos de la misma extensión; 
ademas incluye algunas clases para diseños generales de las paginas a las que esta enlazado. Junto al link de style.css en html se encuentra un link a fontawesome 
el cual permite utilizar los iconos de la pagina mediante unas clases establecidas.

La pagina es una plantilla sencilla de solo 4 documentos html, pero es enteramente responsive; estando asi adaptada para pantallas de dispositivos pequeños, medianos
y grandes.

Espero te guste! ;)
